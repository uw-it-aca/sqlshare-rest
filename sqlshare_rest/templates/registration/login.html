SQLSHARE BETA

SQLShare is a secure and easy way to manage, share, and manipulate your research data. There is no database to install or configure -- just upload your data through a browser, write queries in SQL, and share the results with anyone.

Log in:

Don’t have an account? Create a Google Account and start using SQLShare.

Upload

Just upload any tabular data and start analyzing it instantly in your browser. No hardware, no software to install and configure, no database to design.

Analyze

Write queries, not code. Automate your analysis using full SQL -- joins, nested queries, statistical functions, and more. Derive new virtual datasets and save them on the server for further analysis -- no more jumbles of files polluting your hard drive.

Collaborate

Share your virtual datasets with colleagues by emailing them a link -- no more file attachments disconnected from the script that created them. Your collaborators can derive their own virtual datasets using SQL -- everything is centralized and tracked.
